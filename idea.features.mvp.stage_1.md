# Personal Finance Decision Engine - MVP Feature List

## Core MVP Features

- **Transaction Ingestion**
  - Import CSV / bank statement data
  - Normalize transactions (date, amount, category)

- **Transaction Categorization**
  - Automatic category assignment with confidence scores
  - Allow manual overrides

- **Basic Financial Model**
  - Track recurring vs. discretionary spending
  - Compute cash flow, savings rate, and budget status

- **Decision Engine**
  - Detect overspending in key categories
  - Evaluate one-off purchase requests against current budget and goals
  - Provide clear, structured recommendation (approve / discourage / warn)
